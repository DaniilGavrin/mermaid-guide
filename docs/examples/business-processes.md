# Бизнес-процессы

Моделирование бизнес-процессов с помощью Mermaid.

## 🛒 Процесс покупки

````markdown
```mermaid
graph TD
    A[Клиент заходит на сайт] --> B[Выбирает товар]
    B --> C{Товар в наличии?}
    C -->|Да| D[Добавляет в корзину]
    C -->|Нет| E[Сообщение о недоступности]
    E --> B
    D --> F[Оформляет заказ]
    F --> G{Оплата успешна?}
    G -->|Да| H[Подтверждение заказа]
    G -->|Нет| I[Ошибка оплаты]
    I --> F
    H --> J[Отправка товара]
    J --> K[Получение клиентом]
```
````

**Результат:**
```mermaid
graph TD
    A[Клиент заходит на сайт] --> B[Выбирает товар]
    B --> C{Товар в наличии?}
    C -->|Да| D[Добавляет в корзину]
    C -->|Нет| E[Сообщение о недоступности]
    E --> B
    D --> F[Оформляет заказ]
    F --> G{Оплата успешна?}
    G -->|Да| H[Подтверждение заказа]
    G -->|Нет| I[Ошибка оплаты]
    I --> F
    H --> J[Отправка товара]

```

## 👥 Процесс найма сотрудника

````markdown
```mermaid
sequenceDiagram
    participant HR as HR
    participant Candidate as Кандидат
    participant Tech as Тех.собеседование
    participant Manager as Руководитель
    
    HR->>Candidate: Первичное интервью
    HR->>Tech: Направление на тех.собеседование
    Tech->>Candidate: Техническое интервью
    Tech-->>HR: Результат
    
    alt Успешно
        HR->>Manager: Согласование
        Manager->>Candidate: Офер
        Candidate-->>HR: Решение
    else Отказ
        HR->>Candidate: Уведомление об отказе
    end
```
````

**Результат:**
```mermaid
sequenceDiagram
    participant HR as HR
    participant Candidate as Кандидат
    participant Tech as Тех.собеседование
    participant Manager as Руководитель

    Tech->>Candidate: Техническое интервью
    Tech-->>HR: Результат
    
    alt Успешно
        HR->>Manager: Согласование
        Manager->>Candidate: Офер
        Candidate-->>HR: Решение
    else Отказ
        HR->>Candidate: Уведомление об отказе
    end
```

## 📈 Воронка продаж

Визуализация пути клиента через этапы воронки с использованием User Journey.

**Пример кода:**
```markdown
```mermaid
journey
    title Воронка продаж
    section Узнавание
      Видит рекламу: 5: Пользователь
      Переходит на сайт: 4: Пользователь
    section Интерес
      Просмотр товаров: 3: Пользователь
      Добавление в корзину: 2: Пользователь
    section Покупка
      Оформление заказа: 1: Пользователь
      Оплата: 1: Пользователь
```
```

**Результат:**
```mermaid
