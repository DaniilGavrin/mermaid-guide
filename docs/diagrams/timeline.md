# Диаграммы таймлайн

Диаграммы таймлайн для отображения событий во времени.

## 📐 Базовый синтаксис

````markdown
```mermaid
timeline
    title История проекта
    2020 : Начало
    2021 : Развитие
    2022 : Релиз
```
````

**Результат:**
```mermaid
timeline
    title История проекта
    2020 : Начало
    2021 : Развитие
    2022 : Релиз
```

## 🏗 Практический пример: История IT

````markdown
```mermaid
timeline
    title История вычислительной техники
    1940-е : ENIAC
           : Первый компьютер
    1950-е : Транзисторы
           : Миниатюризация
    1960-е : Интегральные схемы
    1970-е : Персональные компьютеры
           : Apple II, IBM PC
    1980-е : Графические интерфейсы
    1990-е : Интернет
           : Всемирная паутина
    2000-е : Мобильные устройства
           : iPhone, Android
    2010-е : Облачные технологии
    2020-е : ИИ и машинное обучение
```
````

**Результат:**
```mermaid
timeline
    title История вычислительной техники
    1940-е : ENIAC
           : Первый компьютер
    1950-е : Транзисторы
           : Миниатюризация
    1960-е : Интегральные схемы
    1970-е : Персональные компьютеры
           : Apple II, IBM PC
    1980-е : Графические интерфейсы
    1990-е : Интернет
           : Всемирная паутина
    2000-е : Мобильные устройства
           : iPhone, Android
    2010-е : Облачные технологии
    2020-е : ИИ и машинное обучение
```
